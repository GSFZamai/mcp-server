# Sobre

O servidor possui as _tools_ listadas abaixo:

- `getDrinkNames` - Lista nome das bebidas disponíveis;
- `getDrinkDetails` - Descreve os detalhes da bebida;

## Inicializar projeto

O projeto foi desenvolvido utilizando `Typescript` para execução do mesmo deve seguir os passos abaixo:

1. Buildar o projeto com o comando `npm run build`;
2. Executar o projeto com o comando `npm run start`;
